# Sprut Meeting Recorder — автоматическая сборка APK через GitHub Actions

Эта версия настроена так, чтобы **вы не устанавливали ничего на свой компьютер**.
Сборка `.apk` происходит в облаке GitHub — там есть полный доступ к Android SDK,
которого нет у меня в моей рабочей среде.

## Что нужно сделать (5-7 минут, без Android Studio)

### 1. Создайте бесплатный аккаунт на GitHub
Если ещё нет — https://github.com/signup

### 2. Создайте новый пустой репозиторий
На https://github.com/new — назовите как угодно, например `sprut-meeting-app`.
Оставьте его приватным, если не хотите, чтобы код был публичным.

### 3. Загрузите туда содержимое этого архива
Самый простой способ — через веб-интерфейс GitHub:
- Откройте созданный репозиторий
- Нажмите "uploading an existing file" / "Add file" -> "Upload files"
- Перетащите **всё содержимое папки `meeting-app/`** (не саму папку, а её содержимое:
  `www/`, `scripts/`, `.github/`, `package.json`, `capacitor.config.json`,
  `package-android-src/`, `README.md`, `.gitignore`)
- Внизу нажмите "Commit changes"

### 4. Дождитесь автоматической сборки
- Перейдите во вкладку **Actions** вашего репозитория
- Появится запуск "Build Android APK" (запустится сам после загрузки файлов)
- Подождите 5-10 минут, пока не появится зелёная галочка

### 5. Скачайте готовый APK
- Откройте этот завершённый workflow-run
- В самом низу страницы — раздел **Artifacts**
- Скачайте `sprut-meeting-recorder-apk` (это zip с файлом `app-debug.apk` внутри)
- Распакуйте, перекиньте `app-debug.apk` на смартфон (через USB, Telegram
  самому себе, Google Drive — как угодно)

### 6. Установка на S25 Ultra
- Откройте файл `app-debug.apk` на телефоне
- Разрешите "Установка из неизвестных источников", если попросит
- Установите, разрешите доступ к микрофону при первом запуске

## Как перезапустить сборку вручную (если нужно что-то поправить)

Вкладка **Actions** -> workflow "Build Android APK" -> кнопка **Run workflow**
(справа, работает даже без нового пуша в код).

## Проверка на устройстве

1. Нажмите «Начать встречу», заблокируйте экран кнопкой питания на 20-30 сек
2. Разблокируйте — в шторке уведомлений всё это время должна была висеть
   иконка «Идёт запись встречи»
3. Нажмите «Закончить встречу» — запись должна уйти на анализ как обычно

Если Samsung всё же прервал запись — добавьте приложение в исключения
энергосбережения: Настройки -> Батарея -> Ограничения фона -> найти
приложение -> «Не ограничивать».

## Если сборка в Actions упадёт с ошибкой

Откройте упавший шаг в логах Actions, скопируйте текст ошибки и пришлите
её мне — большинство таких ошибок чинится правкой одного файла.

## Что отключено в этой версии

Live-прослушивание руководителем в реальном времени (WebRTC) — только
фоновая запись + отправка на анализ после завершения встречи. Это
сознательный выбор для первого рабочего варианта.
